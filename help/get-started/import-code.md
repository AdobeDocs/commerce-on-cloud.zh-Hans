# 导入现有Commerce代码

Adobe建议您从空白模板或通过导入现有代码在云基础架构项目上创建Commerce。 我们建议先使用空白模板开始，然后将现有代码导入新项目中。 这可确保您拥有支持项目所需的所有文件。

用于导入现有代码的完整工作流包括以下步骤：

- 替换模板代码
- 导入数据库和媒体内容
- 清除缓存并验证导入
